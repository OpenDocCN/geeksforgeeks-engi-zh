# 描述性研究和实验性研究的区别

> 原文:[https://www . geesforgeks . org/描述性研究和实验性研究的区别/](https://www.geeksforgeeks.org/difference-between-descriptive-research-and-experimental-research/)

**1。描述性研究:**
描述性研究是指描述一种现象或正在研究的一个群体的研究，由于操纵变量，在社会科学中很容易进行。它主要用于收集特定人群、情况和事件的数据。描述性研究更倾向于收集数据，并尝试使用统计分析从数据中找出一些见解。

**描述性研究的例子**包括人口普查和产品营销调查等。

**2。实验研究:**
实验研究是指研究者操纵变量得出结论或发现，由于操纵变量而在社会科学中难以做到的研究。这有助于找出偶然关系和关联的因果关系。实验研究也做同样的分析，但它总是分析实验数据的来源。

**实验研究的例子**包括实验室实验。

**描述性研究与实验性研究的区别:**

<center>

| 没有 | 描述性研究 | 实验研究 |
| 01. | 描述性研究是指描述一种现象或一个被研究群体的研究。 | 实验研究是指研究者操纵变量得出结论或发现的研究。 |
| 02. | 由于操纵变量，描述性研究在社会科学中很容易进行。 | 由于操纵变量，实验研究在社会科学中很难进行。 |
| 03. | 它有助于收集特定人群、情况和事件的数据。 | 这有助于找出偶然关系和关联的因果关系。 |
| 04. | 在描述性研究中，研究者观察事物、情况或事件，并描述其中最好的部分。 | 在实验研究中，研究者主要在封闭的环境或实验室中研究事物，并得到最佳的结果。 |
| 05. | 描述性研究不能确定事件的因果关系，因此不能对未来做出预测。 | 实验研究准确地确定了因果关系，因此可以对未来做出预测。 |
| 06. | 主要是描述性研究试图回答“是什么”这个问题。 | 主要是实验研究试图回答“如果”这个问题。 |
| 07. | 描述性研究通常包括社会学和心理学、政治学研究。 | 实验研究通常包括生物学、法医研究和其他实验室研究。 |
| 08. | 它使用定量和定性方法。 | 它主要使用定量方法。 |
| 09. | 描述性研究更倾向于收集数据，并尝试使用统计分析从数据中找出一些见解。 | 实验研究也做同样的分析，但它总是分析实验数据的来源。 |
